# Automated_Door_Lock

## Project Description:
  This project is developed to lock the door of your office, house, locker or anything by using a password. A servo motor is used to control the door locking mechanism.

## Components Used:
  * Arduino Mega 2560 R3
  * 9g Servo Motor
  * 16x2 LCD Display
  * 10K Pot
  * Green LED
  * Red LED
  * 4x4 Membrane Keypad
