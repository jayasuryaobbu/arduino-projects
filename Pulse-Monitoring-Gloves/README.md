# Pulse-Monitoring-Gloves

## Project Description:
  This project is developed to identify the Pulse reading of a human body and to indicate the zone at which the pulse is lying.

## Components Used:
  * Lillypad Arduino
  * Pulse Sensor
  * TM1637 4 Digit 7-Segment Display
  * 3 x AA Batteries
  * Battery Holder
  * Hand Gloves
  * Conductive Thread
